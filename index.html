Halah world!

Halah helps you create reactive graphs to represent X and Y data. View the uncompiled source code or read the documentation at github.com/cheeseen/halah.

Mapping arbitrary numbers to days of the week:

In this example, we are mapping randomly generated X values to the days of the week every second. Notice how halah automagically re-renders all dependant graphs of the model as soon as we use the setX function.

Using Halah, you can arbitarily create multiple charts of the same data model. All graphs that depend on a model will be re-rendered on any data change.

You can easily create different types of chart, currently supported are bar charts and scatter charts.
